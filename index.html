Connect Firebase

Set up a free Firebase Realtime Database so every participant's score instantly appears on this admin panel — regardless of their device. Takes about 3 minutes.

1

Create a Firebase Project

Go to console.firebase.google.com, sign in with a Google account, and click "Create a project". Give it any name (e.g. "QuizEvent").

2

Create a Realtime Database

In the left sidebar, go to Build → Realtime Database. Click "Create Database". Choose any location, then select "Start in test mode". This allows all reads/writes without authentication — perfect for a live event.

3

Register a Web App

Go to Project Settings (gear icon top-left) → General tab → scroll to "Your apps" → click the Web icon (</>). Give it a nickname and click "Register app".
